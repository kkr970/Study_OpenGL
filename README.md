Repository Details
* 사용 프로그램 : VSCode( Visual Studio Code )
* 사용 언어 : C++
* Doc : 공부하면서 작성한 워드 파일
  * 날짜별로 구분 / 큰 주제별로 구분
* src : 소스코드
  * shader : 각 소스코드에 대한 렌더링에 필요한 쉐이더코드
* .exe파일 : src의 코드를 빌드한 결과, 실행 가능
* 나머지 : 외부라이브러리, 텍스쳐, 모델

------------------------
.exe파일 조작키
* 종료버튼
  * ESC : 종료
* 카메라 이동
  * W : 앞
  * S : 뒤
  * A : 왼쪽
  * D : 오른쪽
  * E : 위
  * Q : 아래
* 빛 관련(일부 .exe파일)
  * B : Gamma Correction에서 Blinn on/off
  * SPACE : Shadow Mapping에서 Shadow on/off
  * F : Lighting에서 Spot Light on/off
  * X : 특정 효과의 세기 증가
  * Z : 특정 효과의 세기 감소
